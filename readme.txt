== Description ==

A beautiful and easy customizable set of Gutenberg blocks to list blog posts on your website. Easily adjust the following parameters:

More than 10 blog layout variations
List blog as carousel
Adjust columns number
Set background color
Wide and narrow layout support

== Installation ==

This plugin can be installed directly from your site.

1. Log in and navigate to _Plugins &rarr; Add New_.
2. Type "Kenzap blog‚Äù into the Search and hit Enter.
3. Locate the Kenzap blog plugin in the list of search results and click **Install Now**.
4. Once installed, click the Activate link.
5. You will be redirected to the Kenzap blog Getting Started page.

It can also be installed manually.

1. Download the Kenzap Blog plugin from WordPress.org.
2. Unzip the package and move to your plugins directory.
3. Log into WordPress and navigate to the Plugins screen.
4. Locate Kenzap Blog in the list and click the *Activate* link.
5. You will be redirected to the Kenzap Blog Getting Started page.
